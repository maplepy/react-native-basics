# Tutorial: Creating a movie app with React Native

We will use:
- Core react-native components
  - View
  - Touchable
  - Image
  - FlatList
  - ScrollView
  - SafeAreaView
  - TextInput
  - StatusBar
- Different navigation techniques
  - Stack Navigation
  - Tab Navigation
- Advanced file based routing
  - Dynamic routes
  - Nested routes
- Custom hooks
- Custom fonts
- Types and interfaces with TypeScript (for better type safety)
- Responsive design techniques
- Tailwind CSS for clean and efficient styling
- Backend with Appwrite to handle data and user interactions
- Search feature with optimised performance
- Fetching data from an external API (The Movie Database API)
- Smooth loading and error handling
- A trending movies algorithm based on user interactions
